To Install Service

### This Service For Windows Only

1 - Run Cmd As Administrator
2 - Go To JuhaynaScale.exe Folder
3 - Run This Command
3 - 1. JuhaynaService.exe install
3 - 2. sc config JuhaynaScaleService start=auto
3 - 3. JuhaynaService.exe start

To Uninstall Service

1 - Run Cmd As Administrator
2 - Go To JuhaynaScale.exe Folder
3 - Run This Command
3 - 1. sc JuhaynaScaleService stop
3 - 2. sc JuhaynaScaleService delete
